# obs-covid-cl

Portal que actualiza gráficos descriptivos en base a datos publicados en [Datos-COVID19](https://github.com/MinCiencia/Datos-COVID19).

## Instalación

```
pip install -r requirements.txt
python app.py
```

Luego dirigir navegador a `localhost:8050`.
